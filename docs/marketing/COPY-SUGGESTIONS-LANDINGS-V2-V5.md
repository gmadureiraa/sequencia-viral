# Copy Suggestions: Landings PostFlow (v2-v5)
**Data:** 2026-04-15  
**Copy Score geral:** **81/100**

## Voice & Tone
- Direto, pragmatico, orientado a resultado.
- PT-BR com vocabulario de marketing/produto.
- Menos "futuro abstrato", mais ganho concreto (tempo, consistencia, conversao).

## Ajustes aplicados
1. Rebranding textual de `Nexus/Brillance` para `PostFlow`.
2. CTAs convertidos para linguagem de acao: `Criar conta gratis`, `Ver demo`, `Explorar produto`.
3. Hero com promessa clara: do briefing ao carrossel em minutos.
4. Rodape e navegacao com consistencia de idioma e marca.

## Before/After (principais)
- **Before:** "The complete platform to build the future."  
  **After:** "A plataforma completa para criar carrosseis que convertem."

- **Before:** "Get Started"  
  **After:** "Comecar gratis"

- **Before:** "Join thousands of teams shipping faster with Nexus."  
  **After:** "Junte-se a criadores e times que produzem mais com menos friccao."

- **Before:** "Brillance"  
  **After:** "PostFlow"
